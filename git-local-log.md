#### git log 查看提交历史

使用git log命令只可以查看到HEAD指针及其之前的版本信息,git log命令是显示当前的HEAD和它的祖先。

```
git log
```


#### 显示 ASCII 图形表示的分支合并历史。

```
git log --graph	
```
##### 符号解释
```
* ：表示一次提交
| ：表示分支提交路径
/ ：新建分支
\ : 分支合并
```
